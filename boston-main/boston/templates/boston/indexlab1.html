List of Shiny Applications

1. SciView

The application hosts single cell expression data for different lung studies including healhty, COPD and IPF patients. The application allows browsing the UMAP plots, summarize the multiple gene expressions on cluster maps and browse the pre-computed differential gene expression across cell clusters.

When SciView is properly configured, you'll see a Shiny app above. 

2. Cellxgene

This external application is installed on Minuteman and can be used internally for both restricted as well as public data. The application provides interactive data analyses capabilities. Please contact the data science team for adding the relevant datasets. Links: cellxgene cellxgene (GitHub)

When cellxgene is properly configured, you'll see a the browser snippet above. 

3. pDDExpression

The application hosts cohort expression datasets for interactive browsing and static reporting. Several key COPD and IPF studies are included in the app. Additionally, the app displays several key orthogonal and high-value datasets such as lung genetic association data, GTEx, promoter capture Hi-C, miRNA-binding etc.

When pDDExpressionApp is properly configured, you'll see a Shiny app above. 

4. pDDSingleCell

The application hosts single cell expression data for different lung studies including healhty, COPD and IPF patients. The application allows browsing the UMAP plots, summarize the multiple gene expressions on cluster maps and browse the pre-computed differential gene expression across cell clusters.

When pDDSingleCell is properly configured, you'll see a Shiny app above.
